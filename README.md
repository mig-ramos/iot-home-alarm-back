# iot-home-alarm-back
Residential protection and control system. 

### Cria projeto node
1. npm init -y
2. npm i -D typescript ts-node prisma
3. npx tsc --init
4. "scripts": {
  "start": "ts-node index"
},
5. npx prisma init
6. Para rodar: # npm run
7. Register routes and cors
8. Para rodar arquivos "tsx" qdo salvar: npm install tsx
9. Agora para rodar: npm run dev (mudar script)
10. Gerar a conexão: npx prisma generate
